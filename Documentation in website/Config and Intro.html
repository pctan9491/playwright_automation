Run Mode Attributes

Choose Your Weapon

You can run tests in different modes depending on your immediate goal: Speed, Debugging, or Visual verification.

H

Headless Mode (Standard)

npx playwright test

Fastest. Runs in background. Best for CI/CD.

U

UI Mode (Interactive)

npx playwright test --ui

Best for writing tests. Time-travel debugging & watch mode.

V

Headed Mode (Visual)

npx playwright test --headed

Opens the actual browser window. Watch the robot work.

Understanding the Diagnosis

The report breaks down test execution by browser (Chromium, Firefox, WebKit) and status.

Passed Healthy features working as expected.
Failed Bugs found. Report shows screenshots & traces.
Flaky Unstable tests. Sometimes pass, sometimes fail.
Skipped Tests intentionally not run (e.g., WIP).
